# Conic-Curve-Interactive-Visualizer
Visualize the relationship between e and 3 conic curves


## Features

- Real-time display of circle, ellipse, parabola and hyperbola with continuous eccentricity change
- Bidirectional linkage of e/a/b/c parameters, support slider and number input
- Logarithmic scale for eccentricity, smooth adjustment from 0.0001 to 100
- One-click export of 300DPI high-resolution PNG images

# 圆锥曲线交互可视化工具

高精度圆锥曲线交互可视化工具。

## 功能特性

- 实时展示圆、椭圆、抛物线、双曲线随离心率的连续变化过程
- e/a/b/c四参数双向联动，支持滑块和数字输入两种调节方式
- 离心率采用对数刻度，支持0.0001到100的平滑调节
- 一键导出300DPI高清PNG图片
